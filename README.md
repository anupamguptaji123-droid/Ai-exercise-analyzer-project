### 🎥 Real-Time Camera Analysis
- Detects body landmarks instantly.
- Uses AI pose estimation to monitor exercise movement.
- Displays skeleton keypoints in real time.

### 📊 Session Statistics
Track:

✅ Repetitions (Reps)

✅ Form Quality

✅ Body Stability

### 📐 Joint Angle Monitoring
Measures important body angles:

- 🦵 Left Knee
- 🦵 Right Knee
- 💪 Left Elbow
- 💪 Right Elbow
- 🧍 Spine Angle
- 🦴 Hip Angle

### 💬 Live Feedback
Receive instant suggestions while exercising:

- ✔ Excellent Form
- ⚠ Needs Improvement
- ❌ Unstable Position

### 🔄 Controls
- ▶ Start Camera
- ⏹ Stop Camera
- 🔄 Reset Statistics
- 🏋 Select Exercise

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/AI-exercise-analyzer-project.git
```


## 2️⃣ Move Into Project

```bash
cd AI-exercise-analyzer-project
```


## 3️⃣ Install Dependencies

```bash
pip install flask opencv-python mediapipe numpy
```


## 4️⃣ Start Backend Server

```bash
python app.py
```


## 5️⃣ Open Website

Open:

```text
http://localhost:5501
```

or run using VS Code Live Server.
